
# Check RPG – Flutter Android aplikácia (GitHub Ready)

## Postup:

1. Nahraj tento projekt na GitHub (repozitar: `check-rpg`)
2. GitHub Actions automaticky vytvoria `.apk` po commite
3. .apk nájdeš v sekcii `Actions -> Artifacts -> check-app-release`
4. Uprav backend IP v `main.dart` na IP adresu servera

## Požiadavky:
- Flutter SDK
- Android SDK
- GitHub účet
